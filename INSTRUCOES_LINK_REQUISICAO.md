# Como Solicitar Material do Estoque

## Link de Acesso Rápido

**Abra este link no seu celular ou computador:**

```
https://seu-dominio.com/requisicao-estoque
```

## Passo a Passo

### 1. Abra o Link
- Clique no link acima ou digite no navegador
- Funciona em qualquer celular, tablet ou computador
- Não precisa fazer login!

### 2. Preencha Seus Dados
```
✓ Seu nome completo
✓ Seu WhatsApp (para o estoquista te contatar)
✓ Seu setor (exemplo: Cozinha, Bar, Salão)
```

### 3. Selecione os Estoques
```
De onde? → Escolha de qual estoque você quer pegar
Para onde? → Escolha para onde vai entregar
```

### 4. Adicione os Itens
```
1. Escolha o item da lista
2. Digite a quantidade que você precisa
3. Clique em "Adicionar"
4. Repita para cada item que precisar
```

### 5. Envie!
```
✓ Clique em "Enviar Requisição"
✓ Anote o número da requisição
✓ Aguarde contato do estoquista no WhatsApp
```

## Dicas Importantes

- **Sempre informe o WhatsApp correto** - É por ele que o estoquista vai te avisar
- **Verifique as quantidades disponíveis** - O sistema mostra o que tem em estoque
- **Seja específico nas observações** - Ajuda o estoquista a separar corretamente
- **Guarde o número da requisição** - Facilita o acompanhamento

## Vantagens

- **Rápido** - Não precisa ir até o estoque
- **Prático** - Pode solicitar do celular
- **Organizado** - Sistema registra tudo
- **Eficiente** - Estoquista prepara com antecedência

Problemas com o sistema? Fale com o administrador.
